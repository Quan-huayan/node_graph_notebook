```
应用启动 (main.dart)

## 3. 翻译流程详解

### 调用 i18n.t('Settings')

```
用户代码: Text(i18n.t('Settings'))
    │
    ▼
┌─────────────────────────────────────────────────────┐
│  I18n.t(String key)                                 │
│  ┌─────────────────────────────────────────────┐    │
│  │ 1. 获取当前语言的翻译表                      │    │
│  │    final langMap = _translations[_currentLanguage] │
│  │    // langMap = _translations['zh']          │    │
│  │                                              │    │
│  │ 2. 查找翻译                                  │    │
│  │    return langMap[key] ?? key                │    │
│  │    // langMap['Settings'] = '设置'           │    │
│  │    // return '设置'                           │    │
│  └─────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────┘
    │
    ▼
返回 "设置" (中文) 或 "Settings" (英文)
    │
    ▼
Text Widget 显示翻译后的文本
```

### 翻译数据结构

```
_translations (Map<String, Map<String, String>>)
    │
    ├── 'en' (Map<String, String>)
    │     ├── 'Settings': 'Settings'
    │     ├── 'Home': 'Home'
    │     └── ...
    │
    └── 'zh' (Map<String, String>)
          ├── 'Settings': '设置'
          ├── 'Home': '主页'
          ├── 'AI Tools': 'AI 工具'
          └── ... (400+ 翻译条目)
```

│  │    notifyListeners();  ←─── 触发 Provider   │    │

│  │ 3. 持久化到 SharedPreferences                │    │
│  │    prefs.setString('i18n_language', 'zh')     │    │

    │
    ▼
┌─────────────────────────────────────────────────────┐

    │
    ▼

    │
    ▼
┌─────────────────────────────────────────────────────┐
│  UI 自动更新                                         │
│  - "Settings" → "设置"                               │
│  - "Home" → "主页"                                   │
│  - "AI Tools" → "AI 工具"                            │
│  - 所有文本实时切换为中文                            │
└─────────────────────────────────────────────────────┘
```

## 5. 响应式更新机制

### Consumer<I18n> 工作原理

```
┌─────────────────────────────────────────────────────┐
│  Widget A (使用 Consumer<I18n>)                     │
│  ┌─────────────────────────────────────────────┐    │
│  │ Consumer<I18n>(                               │    │
│  │   builder: (ctx, i18n, child) {               │    │
│  │     return Text(i18n.t('Settings'));           │    │
│  │   }                                            │    │
│  │ )                                              │    │
│  └─────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────┘
    │
    │ 语言切换: i18n.switchLanguage('zh')
    │ notifyListeners() 被调用
    ▼
┌─────────────────────────────────────────────────────┐
│  Provider 检测到 I18n 变化                           │
│  - 标记 Consumer<I18n> 为需要重建                   │
└─────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────┐
│  Widget A 重建                                       │
│  - builder 函数重新执行                              │
│  - i18n.t('Settings') 返回 "设置"                    │
│  - Text 显示 "设置"                                  │
└─────────────────────────────────────────────────────┘
```

### watch 方式 (Flutter 2.8+)

```dart
@override
Widget build(BuildContext context) {
  final i18n = context.watch<I18n>();  // 自动监听

  return Text(i18n.t('Settings'));
  // I18n 变化时，整个 build 方法重新执行
}
```

## 7. 持久化存储机制

### 启动时加载

```
应用启动
    │
    ▼
┌─────────────────────────────────────────────────────┐
│  I18nServiceBinding.createService()                 │
│  final i18n = I18n();                               │
│  i18n.initialize();  // 异步调用                    │
└─────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────┐
│  I18n.initialize()                                  │
│  ┌─────────────────────────────────────────────┐    │
│  │ final prefs = await SharedPreferences     │    │
│  │   .getInstance();                           │    │
│  │                                              │    │
│  │ final savedLanguage = prefs.getString(      │    │
│  │   'i18n_language'                            │    │
│  │ );                                           │    │
│  │ // savedLanguage = 'zh'                      │    │
│  │                                              │    │
│  │ if (savedLanguage != null) {                 │    │
│  │   _currentLanguage = savedLanguage;          │    │
│  │ }                                            │    │
│  └─────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────┘
    │
    ▼
应用显示为上次选择的语言
```

### 切换时保存

```
用户切换语言
    │
    ▼
┌─────────────────────────────────────────────────────┐
│  I18n.switchLanguage('zh')                          │
│  _currentLanguage = 'zh';                           │
│  notifyListeners();                                 │
└─────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────┐
│  持久化到 SharedPreferences                          │
│  ┌─────────────────────────────────────────────┐    │
│  │ final prefs = await SharedPreferences     │    │
│  │   .getInstance();                           │    │
│  │                                              │    │
│  │ await prefs.setString(                       │    │
│  │   'i18n_language',                           │    │
│  │   'zh'                                      │    │
│  │ );                                           │    │
│  └─────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────┘
    │
    ▼
下次启动时自动恢复
```

## 8. 翻译查找算法

```
输入: i18n.t('Settings')
    │
    ▼
┌─────────────────────────────────────────────────────┐
│  步骤 1: 获取当前语言的翻译表                        │
│  final langMap = _translations[_currentLanguage];   │
│  // langMap = _translations['zh']                   │
└─────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────┐
│  步骤 2: 检查语言是否存在                            │
│  if (langMap == null) {                             │
│    return key;  // 语言不支持，返回原文本            │
│  }                                                  │
│  // langMap 存在，继续                               │
└─────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────┐
│  步骤 3: 查找翻译                                    │
│  final translation = langMap[key];                  │
│  // translation = '设置'                            │
└─────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────┐
│  步骤 4: 返回翻译或原文本                            │
│  return translation ?? key;                         │
│  // 返回 '设置'                                      │
│                                                     │
│  如果翻译不存在:                                     │
│  return 'Settings';  // 返回原文本                   │
└─────────────────────────────────────────────────────┘
```

## 10. 性能优化

### 缓存 I18n 实例

```dart
// ✅ 好的做法
class MyWidget extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    final i18n = I18n.of(context);  // 只获取一次

    return Column(
      children: [
        Text(i18n.t('Title')),
        Text(i18n.t('Content')),
        Text(i18n.t('Footer')),
      ],
    );
  }
}

// ❌ 不好的做法
class MyWidget extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        Text(I18n.of(context).t('Title')),      // 重复获取
        Text(I18n.of(context).t('Content')),    // 重复获取
        Text(I18n.of(context).t('Footer')),     // 重复获取
      ],
    );
  }
}
```

### 避免不必要的重建

```dart
// ✅ 使用 Consumer 只重建需要的部分
class MyWidget extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Row(
      children: [
        const Icon(Icons.settings),  // 不变的部分
        Consumer<I18n>(
          builder: (ctx, i18n, child) {
            return Text(i18n.t('Settings'));  // 只重建这个
          },
        ),
      ],
    );
  }
}

// ❌ 整个 Widget 重建
class MyWidget extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    final i18n = context.watch<I18n>();

    return Row(
      children: [
        const Icon(Icons.settings),  // 也会重建（浪费）
        Text(i18n.t('Settings')),
      ],
    );
  }
}
```

## 总结

I18n 插件的实现展示了以下核心概念：

1. **插件化架构**: 国际化作为插件实现，可独立开发和维护
2. **状态管理**: 使用 Provider + ChangeNotifier 实现响应式更新
3. **持久化**: SharedPreferences 保存用户选择
4. **UI Hook**: 通过 Hook 系统集成到工具栏
5. **优雅降级**: 找不到翻译时返回原文本
6. **易于扩展**: 支持动态添加语言和翻译

这个实现简洁而强大，非常适合中小型应用的国际化需求！
